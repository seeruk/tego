# Kitchen Sink

This is a reference example for built-in protobuf and well-known types.

You can use this example to understand what Tego will produce for these types.

Good files to start with:

- `proto/kitchensink/v1/kitchen_sink.proto`
- `kitchensink/kitchen_sink.tego.go`

From this example folder, regenerate with:

```sh
buf generate
```
